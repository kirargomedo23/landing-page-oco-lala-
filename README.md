# Oco LALA Landing Page

¡Bienvenido al repositorio de la página de aterrizaje de Oco LALA!

Esta es una landing page moderna y responsiva, diseñada para mostrar los deliciosos productos de Oco LALA, como frappes, crepes y mini donas. Incluye secciones para el héroe, navegación, menú, pie de página y un botón flotante de WhatsApp para pedidos.

## Tecnologías Utilizadas

-   **React**: Una biblioteca de JavaScript para construir interfaces de usuario.
-   **TypeScript**: Un superconjunto de JavaScript que añade tipado estático.
-   **Vite**: Un empaquetador de módulos rápido para el desarrollo web moderno.
-   **Tailwind CSS**: Un framework CSS de primera clase que te permite construir rápidamente diseños personalizados.
-   **Lucide React**: Un conjunto de iconos para React.

## Características

-   Sección de Héroe atractiva con un mensaje llamativo.
-   Barra de navegación responsiva.
-   Sección de Menú dinámico que muestra los productos.
-   Pie de página con información de contacto y redes sociales.
-   Botón flotante de WhatsApp para facilitar los pedidos.
-   Desplazamiento suave a la sección del menú.
-   Tooltip en el botón de WhatsApp.

## Configuración del Proyecto

Sigue estos pasos para configurar y ejecutar el proyecto en tu máquina local.

### Prerrequisitos

Asegúrate de tener instalado lo siguiente:

-   [Node.js](https://nodejs.org/)
-   [npm](https://www.npmjs.com/) o [Yarn](https://yarnpkg.com/)

### Instalación

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/tu-usuario/oco-lala-landing-page.git
    cd oco-lala-landing-page
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    # o si usas yarn
    # yarn install
    ```

### Ejecución

Para iniciar el servidor de desarrollo:

```bash
npm run dev
# o si usas yarn
# yarn dev
```

Esto iniciará la aplicación en `http://localhost:5173` (o un puerto similar). Abre esta URL en tu navegador para ver la página de aterrizaje.

### Construcción para Producción

Para construir la aplicación para producción:

```bash
npm run build
# o si usas yarn
# yarn build
```

Esto generará los archivos de producción en el directorio `dist/`.
